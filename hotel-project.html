Website Luxury Hotel

Tentang Project

Luxury Hotel merupakan aplikasi berbasis web yang dirancang untuk mempermudah proses reservasi kamar hotel secara online. Pengguna dapat melihat informasi kamar beserta fasilitasnya, melakukan reservasi, serta memantau riwayat pemesanan tanpa harus datang langsung ke hotel. Sistem ini juga dilengkapi fitur admin berupa dashboard untuk mengelola data kamar, data reservasi, dan daftar pengguna secara lebih efisien dan terorganisir.
