Website GuessMyFruit

Tentang Project

GuessMyFruit adalah web yang bisa mengidentifikasi jenis buah hanya dari gambar yang diunggah pengguna. Website ini menggabungkan CNN untuk mengenali buah, Gemini LLM untuk menampilkan informasi nutrisi serta khasiatnya, dan RESTful API sebagai penghubung antar sistemnya. Pengguna cukup mengunggah gambar buah dan sistem akan langsung memberikan hasil prediksi beserta informasi lengkapnya secara real-time.
